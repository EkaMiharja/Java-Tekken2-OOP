# Java Tekken 2 OOP

A simple Java Object-Oriented Programming demonstration that creates a list of fighters from the Tekken 2 game using classes, attributes, and variables.

## Description

This project demonstrates basic Object-Oriented Programming concepts in Java by creating a Fighter class and instantiating multiple fighter characters from the Tekken 2 video game. Each fighter has attributes such as name, fighting style, and hobby.

## Features

- Implementation of a simple `Fighter` class with three attributes
- Creation of multiple fighter objects representing Tekken 2 characters
- Formatted console output displaying fighter information in a table format
- Demonstrates basic OOP concepts: classes, objects, and attributes

## Class Structure

### Fighter Class

The `Fighter` class contains three attributes:
- `name` (String) - The fighter's name
- `fightingStyle` (String) - The fighter's martial arts style
- `hobby` (String) - The fighter's hobby

## How to Run

### Prerequisites
- Java Development Kit (JDK) installed on your system

### Compilation and Execution

1. Clone this repository:
```bash
git clone https://github.com/EkaMiharja/Java-Tekken2-OOP.git
cd Java-Tekken2-OOP
```

2. Compile the Java file:
```bash
javac tekken2.java
```

3. Run the program:
```bash
java tekken2
```
